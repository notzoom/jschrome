Add multiple buttons for bookmarks and bookmarklets by editing this extension. Use the instructions that follow to make a bookmark button. Find this extensions's folder and copy it to the desktop. Replace the folder's name, the url in background.html, the name and title in manifest.json, and the image file labeled icon.png. Delete the line of code containing the key from manifest.json. Go to the extensions tab, enter developer mode, and install the folder unpacked. The default bookmark is the main Google website.
